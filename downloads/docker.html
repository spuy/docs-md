                    # Despliegue para Demo (Docker Compose)

                    Si solo quieres ver nuestros productos puedes hacer un despliegue local de manera rápida con Docker Composeopen in new window.

                    Para hacer el despliegue local puedes descargar esteopen in new window repositorio y seguir las siguientes instrucciones.

                    # Requerimientos

                    Sólo necesitas instalar gitopen in new window y Docker Compose v2.16.0 o superioropen in new window

                    # Clona el repositorio

                    git clone https://github.com/adempiere/adempiere-all-services

                    # Ubica el directorio clonado

                    cd adempiere-all-services

                    # Haz una copia de la plantilla de despliegue

                    cp env_template .env

                    # Levanta el servicio

                    docker compose up
